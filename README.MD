# Chrome扩展工具-二维码生成器

### 起因：
从事移动web开发有段时间了，有的时候往往需要用手机访问网址。开发环境下的URL一般都会超级长，用手机键盘输入确实很蛋疼。

介于通常都是用chrome调试页面，chrome拓展就是最好的解决方案。

于是这个插件横空出世，有以下两大功能：
 
  **1.即刻生成当前页面URL二维码**
 
  **2.支持输入URL转换为二维码**
  
是不是觉得很方便呢？
